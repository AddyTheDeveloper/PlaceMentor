# PlaceMentor - Ultimate Technical Interview Preparation Platform

![PlaceMentor Logo](https://img.shields.io/badge/PlaceMentor-Success-6366f1?style=for-the-badge&logo=icloud&logoColor=white)
![Status](https://img.shields.io/badge/Status-Deployment--Ready-success?style=for-the-badge)

**PlaceMentor** is a premium, modern educational platform designed to help students master **CSE357: Combinatorial Studies** and excel in technical interviews. Built with a focus on deep technical depth, visual learning, and AI-driven interactivity.

## 🚀 Key Features

- **Advanced Technical Curriculum:** In-depth notes for OS, Networking, DBMS, OOP, Data Structures, and Algorithms.
- **Visual Learning Diagrams:** Interactive CSS-based diagrams for complex concepts like OSI Layers, Memory Hierarchy, and Merge Sort.
- **PlaceMentor AI Assistant:** A built-in technical tutor powered by the Gemini API to answer queries instantly.
- **Dynamic AI Quizzes:** Unit-specific MCQ generators that produce unique project-ready questions on demand.
- **Career & Placement Hub:** Dedicated section for Resume building (STAR method), Behavioral Interview strategies, and AI Simulations.
- **Premium UI/UX:** Responsive glassmorphism design, dark/light mode toggle, and smooth animations.

## 🛠️ Technology Stack

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons:** Font Awesome 6.4
- **Fonts:** Google Fonts (Outfit, Inter)
- **AI Integration:** Google Gemini Pro API
- **Persistence:** LocalStorage for progress and theme tracking

## 📱 Mobile Responsive
The platform is fully optimized for all screen sizes, featuring a native-like bottom navigation and high-z-index AI interface for seamless mobile learning.

## 📥 Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/AddyTheDeveloper/PlaceMentor.git
   ```
2. **Open `index.html`** in your preferred browser.
3. **API Setup:** The project comes with a demonstration API key. For production use, replace `GEMINI_API_KEY` in `script.js` with your own from Google AI Studio.
