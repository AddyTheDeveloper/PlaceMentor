    Glossary of Terms

    Kernel: Core of the OS that manages hardware.
    IP Address: Unique identifier for a device on a network.
    Primary Key: Unique identifier for a record in a database table.
    Pointer: Variable that stores the memory address of another variable.
    Time Complexity: Measure of the amount of time an algorithm takes to run.
    Normalization: Process of organizing data in a database to reduce redundancy.
